# Camila Studio

Portfolio profesional de **Camila Studio**, desarrollado con un stack moderno enfocado en rendimiento, diseño y experiencia de usuario.

Este proyecto representa mi perfil como **desarrolladora web frontend** y está pensado tanto para **clientes** como para **recruiters**, mostrando no solo el resultado visual, sino también la forma de trabajar, la estructura del código y las decisiones técnicas.

---

##  Tecnologías principales

* **TypeScript** – Tipado estricto y código mantenible
* **React** – Arquitectura basada en componentes reutilizables
* **Vite** – Entorno de desarrollo rápido y optimizado
* **Tailwind CSS** – Estilos modernos y diseño responsive
* **Git & GitHub** – Control de versiones

---

##  Enfoque del proyecto

El objetivo de este portafolio es comunicar de forma clara:

* Qué hago
* Cómo trabajo
* Qué tecnologías utilizo
* Qué tipo de proyectos desarrollo

Todo el sitio está diseñado con una mentalidad **user-centered**, cuidando accesibilidad, jerarquía visual y rendimiento.

---

##  Estructura del proyecto

```txt
src/
 ├─ assets/        # Imágenes, iconos, avatar, recursos estáticos
 ├─ components/    # Componentes reutilizables (Button, Card, Navbar…)
 ├─ sections/      # Secciones principales del portfolio (Hero, Projects…)
 ├─ pages/         # Páginas si se escala a routing
 ├─ hooks/         # Custom hooks
 ├─ types/         # Tipos y interfaces TypeScript
 ├─ utils/         # Funciones auxiliares
 ├─ data/          # Datos estáticos (proyectos, servicios)
 ├─ styles/        # Configuraciones globales si aplica
 └─ main.tsx       # Entry point
```

Esta estructura permite escalar el proyecto y mantener un código limpio y organizado.
